Project Description

  • Key Components:
    • An AI chatbot designed to interact with PDF documents, answering user queries based on the content within the documents using Retrieval-Augmented Generation (RAG) and LangChain.
    • PDF Parsing: Extracts text from PDF files using libraries like PyMuPDF or pdfplumber.
    • Text Embedding: Converts the extracted text into embeddings using SentenceTransformers
    • Vector Database: Stores the embeddings in a vector database for efficient similarity search. Utilizes LangChain to implement a retrieval system that fetches relevant text chunks from the document based on the user's query.
  • Technologies:
    • LangChain: For chaining together the language models, retrieval mechanisms, and output generation.
    • SentenceTransformers::For generating high-quality text embeddings to represent the PDF content.
    • FastAPi: for creating the endpoints
    • Gemma : an open source LLM from goolge.
    • Features:
      • Contextual Answers: The chatbot provides accurate, context-aware answers
        based on the specific content of the uploaded PDF.
      • Flexible Querying::Users can ask a variety of questions, such as "What is the summary of the introduction?" or "Can you explain the section on machine learning?"
      • Document Understanding:: The chatbot understands complex structures and topics within the document.
